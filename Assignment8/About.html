About

Hello all who visit! This is a short about page to describe why I made this page and more so why I chose this topic for my assignment in WEBD221 for American Public University Systems. For a good time now I have been involved heavily with my health due to my impending doom of some day growing old. Other than that I find weight lifting a time that I can focus on myself and meditate along with spending time with my friends while pushing each other physically. As you can probably tell, I am by no means a web designer but I will try my best to not make everything look the same, website to website that I make. I do however want to be a decent developer so though I am not really worried about design, I love being able to build these sites with less and less repeating and niche code as possible. in this update alone I am able to have 3 HTML pages with a single CSS page linked because I tried my hardest to make everything I write in CSS to be relevant to all the structure not just some niche, independent little element that only needs to be a certain color on a certain page.
